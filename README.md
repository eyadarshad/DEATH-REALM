## Overview

DEATH REALM is a survival horror maze game developed in Unreal Engine 5. Players navigate increasingly complex labyrinths while being hunted by an adaptive AI monster. The game features time-based challenges, multiple hazard types, and five escalating difficulty levels.

---

## Features

- **5 Progressive Levels** - Maze sizes from 10x10 to 30x30
- **Intelligent AI Monster** - Adaptive pathfinding and dynamic speed scaling
- **Multiple Hazards** - Muddy patches, dandelion traps, and safe zones
- **Time Pressure** - Complete objectives before time expires
- **Atmospheric Design** - First-person perspective with flashlight mechanics
- **Adjustable Graphics** - Four quality presets (Low to Ultra High)

---

## Gameplay

### Objective
Navigate mazes to find the Golden Star, unlock the escape cell, and exit before time runs out while evading an AI-controlled monster.

### Controls
- **Movement:** W, A, S, D
- **Look:** Mouse
- **Flashlight:** F
- **Pause:** ESC

### Level Progression

| Level | Size | Features |
|-------|------|----------|
| 1 | 10x10 | Tutorial - No hazards |
| 2 | 15x15 | Monster introduction |
| 3 | 20x20 | Dandelion traps added |
| 4 | 25x25 | Safe zones introduced |
| 5 | 30x30 | All hazards, enhanced difficulty |
